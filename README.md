# 🎵 Spotify Analytics Dashboard | Power BI Project

## 📌 Project Introduction

The Spotify Analytics Dashboard is a Power BI project designed to analyze Spotify music data and uncover meaningful insights about songs, artists, albums, popularity trends, and listening patterns.

This dashboard transforms raw music streaming data into interactive visual analytics that help users explore artist performance, song popularity, album distribution, and audience trends through a modern Spotify-inspired interface.

---

# 🎯 Project Objective

The main goals of this project are:

* To analyze Spotify songs and artist performance efficiently
* To monitor song popularity and ranking trends
* To identify top-performing artists and tracks
* To compare album types and music trends
* To create an interactive and visually appealing music analytics dashboard

* Taylor Swift ranks among the top artists in popularity and song count.
* Album-type songs dominate the dataset compared to singles and compilations.
* Explicit songs represent a significant portion of tracks.
* Song popularity fluctuates throughout different months.
* Several artists consistently produce high-ranking hit songs.
